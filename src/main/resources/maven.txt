clean- очистка артефакта (удаляем папку target)
validate- проверка структуры, всё ли корректно(mvn clean validate(в терминале))
compile- компиляция java кода в .class
test- запускает unit test
package- упаковка в .jar
verify- проверка все ли тесты выполнились успешно
install- установка приложения (артефакта) в локальный репозиторий
site- генерация документации
deploy- отправка готового артефакта на удалённый репозиторий

mvn dependency:tree просмотр дерево зависимостей
mvn dependency:purge-local-repository
mvn help:effective-pom
mvn clean verify -DskipTests
mvn surefire:report
mvn spring-boot:run
